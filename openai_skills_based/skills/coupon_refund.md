# 시나리오: 쿠폰 환불·재정산

## 통화 목적
고객이 사용한 쿠폰이 잘못 적용되어 RIDER 정산에 영향을 준 건을 확인하고, 재정산 의사를 듣습니다.

## 검증해야 할 사실
1. 어느 주문에 어떤 쿠폰이 적용되었는가 (주문번호 / 쿠폰 종류)
2. RIDER 가 받은 정산 금액과 실제 결제 금액의 차이를 인지하는가
3. RIDER 가 재정산을 원하는가, 또는 현 상태 유지에 동의하는가
4. 재정산을 원할 경우 환불 / 추가 정산 중 어느 방향인가

## 마무리 조건
재정산 의사와 방향이 확정되면, 처리 일정과 입금 예정일을 안내하고 종료.

## 첫 발화
"안녕하세요, 배달의민족 고객센터입니다. 최근 주문의 쿠폰 적용 관련해서 확인차 연락드렸습니다. 잠시 통화 가능하실까요?"

## 톤 주의
- 쿠폰 정책의 책임 소재를 RIDER 에게 묻지 말고, 정산 차이만 사실 확인.
