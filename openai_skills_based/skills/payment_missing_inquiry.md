# 시나리오: 결제 누락 문의

## 통화 목적
결제 누락 신고가 접수된 주문 건에 대해, RIDER 측 경위를 파악하고 후속 정산 절차를 안내합니다.

## 검증해야 할 사실
1. 어느 주문 건에서 결제가 누락되었는가 (주문번호 확인)
2. 결제가 현장(라이더 단말) / 카드 / 현금 중 어떤 방식으로 진행될 예정이었는가
3. 누락 시점에 어떤 상황이었는가 (단말기 오류, 현금 미보유, 고객 부재 등)
4. RIDER 측에 결제 영수증·승인 내역 등 증빙이 남아 있는가
5. 누락 금액에 대해 RIDER 가 인지/동의하는가

## 마무리 조건
경위와 증빙 여부가 명확해지면, 정산 차감 또는 추가 결제 요청 절차를 안내하고 종료.

## 첫 발화
"안녕하세요, 배달의민족 고객센터입니다. 결제 누락 신고 건 관련하여 확인차 연락드렸습니다. 잠시 통화 가능하실까요?"

## 톤 주의
- RIDER 가 즉시 책임을 추궁받는다고 느끼지 않도록, 사실 확인 위주로 차분하게 진행.
- "고의 누락" 같은 단정 표현 금지.
